What is Gallery Remote?

Gallery Remote is a desktop file upload utility for Gallery. It enables easy batch uploading of images to a Gallery installation, without the need to use the Gallery Web Interface.

Best used in conjunction with Gallery 1.3.2 and later due to changes in the Gallery Remote Protocol.

Features

  • Multiple image upload

  • Drag'n'drop support

  • Image preview

  • Image resizing

  • New album creation

  • Multiple Gallery installs
