Importing into gallery

Gallery has the ability to import (captions, descriptions, etc) from external formats. Importing is currently only available at upload time. Form and URL uploads are supported.

  • When uploading by form, follow steps 3 and 4. Specify the number of meta data files in step 3. Specify the meta data files in step 4, these files' data will be applied to the newly added items.

  • When uploading by URL, place the meta data files in the same directory as the images. When gallery reads the URL(s), it will list the images, movies, and other support formats for your selection. Below, it will list any meta data files that were found, if any. Only the selected meta data files will be read and applied to the newly added items.

Meta data file requirements:

  • For .csv files, a text file with semicolon seperated fields. The first line (or row) defines each column. A column of "File Name" or "Filename" must exist, this field name is not case sensitive. All other field names are case sensitive. The remainder of the fields will be used to populate the custom fields. If the custom fields are not set, then they will be ignored. If no caption field is specified, the first of the following non empty fields will be used: Title, Description, Persons. (in order)
