Getting Additional Help

If for some reason, you cannot get Gallery installed, don't worry. Gallery has an excellent support network, and we'll get you up and running!

Here's a list of things to do if you get stuck.

  • Read the Gallery FAQ. The FAQ is constantly updated with new information. Your question is probably answered in the FAQ.

  • Read the Gallery User Guide. The User Guide contains help regarding many problems.

  • If all else fails, ask in the Gallery Forums. Please make sure you have read the FAQ and User Guide throughly before asking here. Also, please make sure you pick the most appropriate forum, or your post will be moved.
