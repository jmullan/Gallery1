Gallery 2 Release Candidate 2 (kthxbye) Released!

This is the second release candidate of Gallery 2.  If all goes well, the next release will be the final version.  The purpose of this release candidate is to let the product cool for a couple of weeks so that we can identify any fix any last minute issues that you may discover.  You should download and use it as if it were the final release.

We have made <a href="http://tinyurl.com/cnklu#whats_new">small improvements</a> since the RC-1 release.  As with RC-1, we have 4 different packages for you to choose from.  You can read all about it on our <a href="http://codex.gallery2.org/index.php/Gallery2:Download">Gallery 2 Download Page</a>.

As always before you <a href="http://tinyurl.com/5gnbs">ask for help</a>, please, please <a href="http://tinyurl.com/cnklu">read the README</a> -- we've made an effort to answer all of your questions there.  Especially read the <a href="http://tinyurl.com/cnklu#known_issues">known issues</a> section.  If you're upgrading, be sure to scan the super easy <a href="http://tinyurl.com/cnklu#updating">updating instructions</a> first.  Then head on over to the <a href="http://tinyurl.com/3rgzx">G2 Forums</a> and let us know what you think!

Thanks for your continued support!
